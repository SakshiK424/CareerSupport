# 🎓 AI Career Mentor Agent

An AI-powered Career Mentor built using **Streamlit** that provides personalized career guidance, resume analysis, skill-gap identification, and learning recommendations.

This project was developed as part of the **IBM SkillsBuild AI Internship**.

---

## 📌 Problem Statement

Many students struggle to:
- Choose the right career path.
- Identify missing skills for their dream job.
- Improve their resumes.
- Find suitable learning resources.
- Prepare for placements effectively.

---

## 💡 Solution

The AI Career Mentor helps students by:

- 📄 Analyzing resumes
- 🎯 Suggesting suitable career paths
- 📚 Identifying skill gaps
- 🚀 Recommending courses and certifications
- 💼 Providing placement preparation guidance
- 📝 Offering resume improvement suggestions

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/skillsbuild.git
```

Move into the project folder

```bash
cd skillsbuild
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
